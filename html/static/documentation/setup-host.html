SPACE Host Setup

This guide will demonstrate how to setup Space on a remote server, such as a Digital Ocean Droplet running Ubuntu 18.04 x64.

Access

Create a new droplet and ssh into the IP address


                ssh root@your_server_ip

Create a new user in the sudoers group


                # Create space user
                adduser your_server_username

                # Add space user to sudoers
                usermod -aG sudo your_server_username

Configure DNS for your_server_domain to your_server_ip

Firewall

Install a firewall such as UFW and open the ports used by HTTP, HTTPS, and Space


                # Install firewall
                apt install ufw

                # Allow http port
                ufw allow 80

                # Allow https port
                ufw allow 443

                # Allow space ports
                ufw allow 22222
                ufw allow 22322
                ufw allow 23232

                # Enable firewall
                ufw enable

HTTPS

Use a Certificate Authority such as Let's Encrypt to generate a certificate


                # Install certbot
                apt install certbot

                # Generate certificate
                certbot certonly --standalone -d your_server_domain

Cron

Cron can be used to trigger certificate renewal


                # Open cron editor
                sudo crontab -e

                # Renew certificate on the first day of the week
                * * * * 0 sudo certbot renew --pre-hook "systemctl stop space" --post-hook "systemctl start space"

Payment Processor

Create an account with a Payment Processor such as Stripe, and add two subscriptions (Mining + Storage).

Space

Create a directory


                # Create space directory
                mkdir -p /home/your_server_username/space/

Build Space and copy the files from your development machine to your server


                # Copy server binary
                rsync $GOPATH/bin/spaceservergo-linux-amd64 your_server_username@your_server_ip:~/space/

                # Copy website content
                rsync -r $GOPATH/src/github.com/AletheiaWareLLC/spaceservergo/html your_server_username@your_server_ip:~/space/

                # Copy client binaries into website static content
                rsync $GOPATH/bin/spaceclientgo-* your_server_username@your_server_ip:~/space/html/static/

Initialize Space with your_server_domain as the alias


                ALIAS=your_server_domain ROOT_DIRECTORY=~/space/ ~/space/html/static/spaceclient-linux-amd64 init

                # Allow spaceservergo to read security credentials
                chown -R your_server_username:your_server_username /etc/letsencrypt/

                # Allow spaceservergo to bind to port 443 (HTTPS)
                # This is required each time the server binary is updated
                setcap CAP_NET_BIND_SERVICE=+eip /home/your_server_username/space/spaceservergo-linux-amd64

Configure Space with your stripe information, alias, password, root directory, certificate directory, and list of peers


                # Create space config
                cat > /home/your_server_username/space/config <STRIPE_PUBLISHABLE_KEY=VVVVVV
                >STRIPE_SECRET_KEY=WWWWWW
                >STRIPE_PRODUCT_ID=XXXXXX
                >STRIPE_PLAN_ID=YYYYYY
                >STRIPE_WEB_HOOK_SECRET_KEY=ZZZZZZ
                >ALIAS=your_server_domain
                >PASSWORD='VWXYZ'
                >ROOT_DIRECTORY=/home/your_server_username/space/
                >CERTIFICATE_DIRECTORY=/etc/letsencrypt/live/your_server_domain/
                >PEERS=space.aletheiaware.com,bc.aletheiaware.com
                >EOF

systemd

Setup Space as a service which is started at boot


                # Create space service
                cat > /etc/systemd/system/space.service <[Unit]
                >Description=Space Server
                >[Service]
                >User=your_server_username
                >WorkingDirectory=/home/your_server_username/space
                >EnvironmentFile=/home/your_server_username/space/config
                >ExecStart=/home/your_server_username/space/spaceservergo-linux-amd64
                >SuccessExitStatus=143
                >TimeoutStopSec=10
                >Restart=on-failure
                >RestartSec=5
                >[Install]
                >WantedBy=multi-user.target
                >EOF

                # Reload daemon
                systemctl daemon-reload

                # Start service
                systemctl start space

                # Stop service
                systemctl stop space

                # Monitor service
                journalctl -u space

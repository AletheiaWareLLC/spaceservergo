SPACE Usage

Environment

  • $> ALIAS=alice ./space

    Sets the alias for Space to load from the keystore.

  • $> ROOT_DIRECTORY=/tmp/foobar ./space

    Sets the root directory for Space to use.

  • $> CACHE_DIRECTORY=/tmp/cache ./space

    Sets the directory for Space to use for caching.

  • $> DEBUG=true ./space

    Sets the debug flag for Space.

  • $> KEYS_DIRECTORY=~/keys ./space

    Sets the directory for Space to use for storing keys.

  • $> CERTIFICATE_DIRECTORY=~/certificates ./space

    Sets the directory for Space to use for storing certificates.

  • $> LOG_DIRECTORY=/tmp/logs ./space

    Sets the directory for Space to use for logging.

  • $> PASSWORD=password1234 ./space

    Sets the password for Space to unlock the private key from the keystore.

    Note: This is NOT advisable as your password will be stored in the command history, it is more secure to enter your password at the prompt.

  • $> PEERS=space.aletheiaware.com,foobar.com ./space

    Sets the peers for Space to communicate with.

Commands

  • $> ./space

    Displays usage.

  • $> ./space init

    Initializes environment, generates keys, and registers alias.

  • $> ./space add [name] text/plain

    Adds a new file with the given name, mime type, and contents read from stdin.

  • $> ./space add [name] text/plain foobar.txt

    Adds a new file with the given name, mime type, and contents read from file.

  • $> ./space add-remote foo text/plain

    Add a new file with the given name, mime type, and contents read from stdin using a Remote Mining Service to save your device battery.

  • $> ./space add-remote foo text/plain foobar.txt

    Add a new file with the given name, mime type, and contents read from file using a Remote Mining Service to save your device battery.

  • $> ./space list

    Displays a list of files in reverse-chronological order.

  • $> ./space show [hash]

    Displays the metadata of the file with the given hash.

  • $> ./space showall text/plain

    Shows all files with the given mime type.

  • $> ./space get [hash]

    Gets file with the given hash and writes the data to stdout.

  • $> ./space get [hash] file

    Gets file with the given hash and writes the data to the given file.

  • $> ./space tag [hash]

    Displays the tags (if any) of the file with the given hash.

  • $> ./space share [hash] alice bob

    Shares the file with the given hash with the given aliases.

  • $> ./space tag [hash] foo bar

    Adds the tags to the file with the given hash.

  • $> ./space search foo bar

    Displays the metadata of all files which match the given tags.

  • $> ./space preview [hash]

    Displays the preview (if any) of the file with the given hash.

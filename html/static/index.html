Space

Welcome to Space, please follow the steps below to get setup;

  1. Download

    Get Space for your device;

    • Android - Google Play Store or APK
    • iOS - coming soon!
    • JVM - JAR
    • Linux - ARM (v5, v6, v7, v8) or Intel/AMD (32, 64)
    • Mac - Intel/AMD (32, 64)
    • Windows - Intel/AMD (32, 64)

  2. Initialize

    Generate your key pair consisting of a public key (which will be registered with your account) and a private key (which does not leave your possession).

    $> ./space init
    Creating keystore under /Users/username/bc
    Enter keystore password:
    Confirm keystore password:
    Generating RSA 4096 Public/Private Key Pair
    Successfully Created Key Pair
    ...
    ...
    Initialized as username

  3. Sample Usage

    Environment 

    • $> ALIAS=alice ./space

      Sets the alias for Space to load from the keystore.

    Commands 

    • $> ./space [name] text/plain

      Adds a new file with the given name, mime type, and contents read from stdin.

    • $> ./space [name] text/plain foobar.txt

      Adds a new file with the given name, mime type, and contents read from file.

    • $> ./space get [hash]

      Gets file with the given hash and writes the data to stdout.

    • $> ./space get [hash] file

      Gets file with the given hash and writes the data to the given file.

    • $> ./space init

      Initializes environment, generates keys, and registers alias.

    • $> ./space list

      Displays a list of files in reverse-chronological order.

    • $> ./space show [hash]

      Displays the metadata of the file with the given hash.

    • $> ./space showall text/plain

      Shows all files with the given mime type.

    • $> ./space tag [hash]

      Displays the tags (if any) of the file with the given hash.

    • $> ./space tag [hash] foo bar

      Adds the tags to the file with the given hash.

    • $> ./space search foo bar

      Displays the metadata of all files which match the given tags.

    • $> ./space preview [hash]

      Displays the preview (if any) of the file with the given hash.

    • $> ./space share [hash] alice bob

      Shares the file with the given hash with the given aliases.

    • $> ./space customer

      Displays Stripe customer information.

    • $> ./space subscription

      Displays Stripe subscription information.

    • $> ./space upload foo text/plain

      Uploads a new file with the given name, mime type, and contents read from stdin to Aletheia Ware's Remote Mining Service, saving your device battery.

    • $> ./space upload foo text/plain foobar.txt

      Uploads a new file with the given name, mime type, and contents read from file to Aletheia Ware's Remote Mining Service, saving your device battery.

  4. Remote Mining Service

    Save battery by mining remotely, register your alias and billing information using; 

    $> ./space subscription
    or click Subscription and provide the following information;
    • Alias - registered to your public key and signed by your private key.
    • Email - how you would like us to contact you.
    • Billing Info - how you would like to pay (this information is handled and stored by Stripe);
      • Card Holder Name
      • Card Number
      • Card Expiry Date
      • Card Verification Code (CVC)
      • Billing Address
